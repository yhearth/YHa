A new Leap

( 성장배경 )

첫 꿈은 개발자는 아니었습니다.디자인이 좋아 예술고등학교 디자인을 전공했고 디자이너라는 목표로 계원예술대학교 디지털미디어디자인에 들어오게 되었습니다. 대학교에서는 디자인뿐만이 아닌 기획과 프로그래밍 모두 배울 수 있었습니다. 전공수업으로 들었던 프로그래밍 수업은 첫 웹 사이트를 만든 후에는 웹 퍼블리셔 라는 목표로 꿈을 바꾸게 되기에 충분했습니다.

처음 만든 웹 사이트는 html과 css 으로만 사용한 작업이었지만 웹을 디자인만 하는 것을 넘어 웹 페이지를 웹 브라우저에 동작할 수 있도록 하고 사용자 경험을 고려한 다양한 기능을 구현하는 것에 큰 성취감을 얻었습니다. 새로운 언어를 이해하고 기술을 익히면서 지식을 확장해 나가는 것을 경험할 수 있었습니다. 이 경험으로 프로그래밍 과목 성적에서 A라는 결과를 받을 수 있었고 웹 퍼블리셔라는 새로운 도전에 첫 발을 내딜 수 있었습니다.

A leap for
Tomorrow's Experience

( 직무경험 )
소통에서 나오는 가치가 만들어낸 솔루션

졸업 작품 ( repick! ) 에서는 기획자,디자이너와 협업하여 만드는 팀 작업이였습니다 팀 작업이다 보니 프로젝트를 바라보는 서로 다른 방향들을 조율하기 쉽지 않았지만 교수님과의 피드백과 지속적인 커뮤니케이션 으로 여러 문제을 해결할 수 있었습니다. 각자 다른 시선에서 주고받는 의견들은 긴밀한 유대감을 형성시켜 프로젝트를 더욱 완성도 있게 만들 수 있었고 혼자가 아닌 같이 일하는 방법을 터득하게 되었습니다. 이런 커뮤니케이션은 개발자로서 저를 한 번 더 성장할 수 있게 만들어 주었습니다. 개발자들과의 커뮤니티에 참여하고 코드를 리뷰 받으며 의견을 수렴하는 과정이 더 나은 코드를 작성할 수 있게 만들었고 개발자로서 좋은 경험이 된 작업이였습니다. 존중과 소통은 프로젝트 성공에 큰 기여를 할 수 있었고 개발은 단독으로 하는 것이 아닌 협업과 소통으로 좋은 결과를 얻을 수 있다는 걸 배울 수 있었습니다

새로운 경험을 위해 고민의 결과

졸업 후에는 웹 퍼블리셔의 목표에서 더 나아가 프론트 엔드로 목표로 넓혔습니다. 더 성장하기 위해 계속해서 여러 기술을 찾아보고 더 나은 웹을 위해 배움을 이어 나갔습니다. 다양한 웹 레이아웃을 구현하기 위해 swiper 와 gsap 같은 라이브러리를 공부했습니다. 라이브러리를 이용한 ( HMG ) ( TAMBURINS ) 는 기존의 작업보다 다양한 시각적인 효과를 제공할 수 있었고 사용자가 웹 경험이 더욱 흥미롭도록 만들기 위해 노력했습니다. 라이브러리 없이도 구현할 수 있도록 javascript으로만 작업한 ( FRITZ HANSEN ) 사이트는 필터링 기능과 커스텀 기능을 구현하기 위해 많이 문제들을 직면했지만 하나하나 분석하고 해결하는 과정 속에서 유연한 사고를 학습할 수 있었습니다. 지금까지 해왔던 프로젝트 경험을 통해 다양한 기술과 경험, 창의적인 해결책을 떠올리는 데 도움이 되었고 계속해서 새로운 기술에 대한 호기심과 열정을 가지고 스스로 역량을 성장 시키고 싶습니다 .

현재는 웹 사이트에서 프로그래밍으로 다양한 그래픽을 표현할수 있도록 webGL 라이브러리인 three.js 를 배우고 있습니다. Canvas, WebGL에 관심이 많아 찾아보고 공부하고 있습니다.

Attitude for a leap
